### 1.2 BondPool 合约

**作用：** Agent 存入质押代币 → 交易上限 = 质押量 × 10。

```
质押 100 USDC → 单笔交易上限 1,000 USDC
质押 0 USDC  → 交易上限 0（Agent 无法操作）
```

**接口：**

| 函数 | 说明 |
|---|---|
| `deposit(agent, amount)` | 存入质押，增加交易额度 |
| `withdraw(agent, amount)` | 取出质押，减少交易额度 |
| `getTxLimit(agent)` | 查询 Agent 单笔交易上限 |
| `slash(agent, amount, recipient)` | 罚没质押（Owner only）|

### 1.3 CircuitBreaker 合约

**作用：** 监控 Agent 的 L1 校验失败频率。连续 N 次 FAIL → 自动暂停。

**默认参数：**

| 参数 | 默认值 | 说明 |
|---|---|---|
| `defaultThreshold` | 5 | 连续 FAIL 次数触发暂停 |
| `defaultCooldown` | 86400s (24h) | 暂停冷却时间 |
| `dailyTxValue` | 按日累计 | 日交易金额超限 → 暂停 |

**状态转换：**

```
[ACTIVE] ── recordFail() × 5 ──→ [PAUSED]
[ACTIVE] ── recordSuccess() ──→ [ACTIVE] (计数器归零)
[PAUSED] ── resume() ──→ [ACTIVE]
```

### 1.4 Chiron.sol 集成

```
storeReceipt()
  │
  ├─ circuitBreaker.isPaused(agent) → true → revert("Agent is paused")
  │
  ├─ l1Result == FAIL → circuitBreaker.recordFail(agent)
  ├─ l1Result == PASS → circuitBreaker.recordSuccess(agent)
  │
  └─ 正常存证
```
